# 🚀 Vaibhav Khator - Personal Portfolio Website

## 📖 Description

A modern, responsive, and visually engaging personal portfolio website designed to showcase my technical skills, academic journey, and professional projects. Built with a focus on clean UI/UX principles, this static website highlights my expertise in software development, AI/ML concepts, and data management.

## ✨ Features

  * 📱 **Fully Responsive Design:** Optimized for seamless viewing across desktops, tablets, and mobile devices.
  * 🎨 **Modern Aesthetics:** Utilizes a sleek dark mode theme with glassmorphism effects and elegant typography.
  * ⚡ **Smooth Animations:** Includes smooth scrolling, subtle fade-ins, and interactive hover effects to enhance user experience.
  * 🗂️ **Well-Organized Sections:** Dedicated areas for About Me, Skills, Projects, Experience, and Education.
  * 📬 **Interactive Contact UI:** A fully styled front-end contact form with visual feedback simulation.

## 🛠️ Tech Stack

  * **Markup:** HTML5
  * **Styling:** CSS3 (Flexbox, CSS Grid, Custom Variables)
  * **Interactivity:** Vanilla JavaScript (ES6+)
  * **Typography & Icons:** Google Fonts (Poppins), FontAwesome 6

## 🚀 Installation & Setup

Since this is a static website built with standard web technologies, no complex build tools or dependencies are required.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/vaibhavkhator/portfolio-website.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd portfolio-website
    ```
3.  **Run locally:**
    Simply open the `index.html` file in your preferred web browser, or use an extension like VS Code Live Server for hot reloading.

## 💡 Usage

Users can visit the site to:

  * Read my professional summary and academic background.
  * Review my technical skills visualized with progress bars.
  * Explore my featured projects, including descriptions and tech stacks.
  * View my professional timeline and certifications.
  * Use the contact form to reach out directly.
